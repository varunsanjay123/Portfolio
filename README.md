# Varun Sanjay - Space Themed Portfolio

A premium, interactive, and high-performance portfolio built with React, Framer Motion, and Three.js. This project features a stunning 3D space environment with parallax effects, dynamic typing animations, and a sleek glassmorphism UI.

## 🚀 Live Demo
[View Live Site](https://portfolio-seven-xi-89.vercel.app/)

## ✨ Features
- **Stellar 3D Background**: Interactive Three.js scene with planets (including Saturn), stars, and parallax movement.
- **Custom Space Cursor**: A glowing stellar cursor with nebula auras and trailing effects.
- **Dynamic Typography**: Character-by-character staggered animations and typewriter effects.
- **Glassmorphism UI**: Modern transparent cards with backdrop blurs and subtle glows.
- **Smooth Navigation**: One-page scrolling with active section tracking.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports.
- **Project Filtering**: Interactive category filtering for Development and Data Analytics projects.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite
- **Animations**: Framer Motion
- **3D Graphics**: Three.js, React-Three-Fiber, React-Three-Drei
- **Styling**: Tailwind CSS 4
- **Icons**: React Icons

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/varunsanjay123/Portfolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
